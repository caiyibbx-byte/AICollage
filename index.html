学院简介
🤖
广州城市理工学院 · 人工智能产业学院
GCUT AI INDUSTRY COLLEGE

人工智能产业学院是广州城市理工学院响应国家人工智能发展战略，深度融合人工智能技术与产业应用打造的新型产教融合平台。学院立足粤港澳大湾区数字经济与智能产业发展需求，联合麒麟软件、龙芯中科、盖特软件、广东派宝机器人等头部企业深度开展产学研合作，引入企业完整的技术链、项目链与服务链，推行双导师制、项目化教学，打通校园学习、岗位实训、高质量就业全链条。

学院打破传统专业壁垒，依托五大核心方向构建"技术研发—场景应用—运营管理—创新创业"全产业链培养体系，面向企业数智化运营、智慧城市、国产芯片、信创生态、无人自主系统等战略性新兴领域，培养具备AI思维、工程能力与行业洞察力的高素质复合型人才，致力于打造粤港澳大湾区AI人才高地与产业发展引擎。

核心合作企业

学院联合国产软硬件与智能技术领域头部企业，共同构建产教融合生态，为学员提供真实项目资源、企业导师与就业通道。

Part/02
招生总体说明
01 / 招生宗旨

坚持产业导向、精准分流、分类培养原则，结合人工智能技术发展趋势与行业岗位需求，匹配不同专业学生特长，补齐单一专业技能短板，深化产教融合，为企业数字化运营、智慧城市建设、国产芯片应用、信创生态开发、无人自主系统等领域输送复合型AI专业人才。

02 / 招生对象

本次面向广州城市理工学院全体大二在读本科生，无专业、文理类别限制，欢迎所有对人工智能感兴趣的同学报名参与。

03 / 报名基本条件
  • 1、
    热爱人工智能及相关产业，学习态度端正，自律性强，具备良好的自主学习与动手实践能力；
  • 2、
    积极参与课程学习、项目实训、企业实践，服从产业学院教学、实训及日常管理规定；
  • 3、
    遵守校纪校规，无违纪记录，能够保证全程参与培养、实训及考核；
  • 4、
    具备基本编程基础（Python / C / Java 之一）者优先，无基础但有强烈学习意愿者同等欢迎报名。
Part/03
五大培养方向

结合专业属性与行业岗位，设置五大差异化培养方向，所有课程均可执行学分置换，不影响学生本专业正常学业。

01
企业数智化运营方向
培养目标掌握AI驱动的企业数字化转型方法论，能够运用大模型、RPA、数据分析等工具进行业务流程优化与智能决策支持，培养具备"业务+AI"双重能力的数智化运营人才
核心课程制造业数字化认知与业务建模、企业数据工程与AI服务产品、智能排产与生产过程数字化、供应链协同与质量智能检测、设计数字化与客户协同平台、企业云服务技术与运维、模型训练和调优
适配岗位AI产品经理、数字化转型顾问、智能运营专员、大模型应用开发工程师
02
智慧城市与数智化服务方向
培养目标面向政务数字化、智慧园区、智慧交通、公共安全等城市级AI应用场景，培养能够参与政府数字化项目建设、智慧城市解决方案设计与交付的复合型工程人才
核心课程智能体与OPC实践、无人自主系统与实践、模型训练和调优；场景方向（6选2）：智慧教育平台与实施、家政服务数智化管理、楼宇智能化系统与应用、农业数智化技术与应用、文旅数智化应用、城市数据工程与AI应用；视听方向（2选1）：智能视听系统基础、视听方案设计与工程表达
适配岗位智慧城市解决方案工程师、AI系统集成工程师、政务信息化项目专员、数字孪生开发工程师
03
龙芯芯片设计与应用开发方向
培养目标聚焦国产LoongArch架构芯片平台，培养具备国产处理器底层开发、嵌入式系统移植与AI推理部署能力的工程人才，服务国家芯片自主可控战略
核心课程硬件描述语言与数字系统设计、汇编语言、EDA技术、LoongArch体系结构与系统优化、编译技术与代码适配、智能硬件设计与应用开发
适配岗位国产芯片应用开发工程师、嵌入式系统工程师、底层软件工程师、AI推理优化工程师
04
麒麟开源社区开发与信创方向
培养目标围绕国产操作系统麒麟（Kylin OS）生态，培养能够承担国产软硬件适配、信创平台应用开发与社区贡献的实战型开发人才，服务党政军企信创替代工程
核心课程Linux内核开发与社区实践、Linux系统原理与麒麟OS架构、麒麟平台系统服务与安全加固、编译技术与应用适配、基于麒麟操作系统的智能系统开发、麒麟平台应用开发
适配岗位信创系统工程师、国产软件适配工程师、开源社区开发者、Linux运维工程师
05
无人自主技术方向
培养目标面向无人机、无人车、机器人等无人自主系统领域，培养具备感知算法、运动规划、嵌入式控制与ROS2开发能力的复合型自主系统工程人才
核心课程机器人综合基础、无人系统算法、自动器调度管理平台技术、具身智能与大模型技术、无人应用开发
适配岗位无人机/无人车算法工程师、机器人系统开发工程师、自主感知工程师、ROS2开发工程师
Part/04
五大核心培养板块

学院构建"AI基础—技术深化—行业应用—项目实战"四层课程体系，覆盖从算法理论到工程落地的全流程，理论与实战深度融合。

🧠 01 AI基础与大模型应用板块
培养方向Python编程、机器学习基础、深度学习、大模型原理与调用、Prompt工程
实训内容HuggingFace模型调用实战、RAG系统搭建、LangChain/LangGraph多智能体开发
适配岗位AI工程师、大模型应用开发工程师、AI产品经理
🏙️ 02 智能系统与行业落地板块
培养方向计算机视觉、视频智能分析、物联网数据处理、行业AI方案设计
实训内容YOLO目标检测部署、视频智能分析系统实操、智慧城市AI项目实训
适配岗位AI系统集成工程师、算法工程师、行业AI解决方案工程师
🔧 03 国产软硬件与信创板块
培养方向LoongArch架构开发、麒麟OS应用、国产数据库、信创软件适配与迁移
实训内容龙芯开发板实战、麒麟OS上的AI应用开发、国产数据库实操、国产中间件适配
适配岗位信创工程师、嵌入式开发工程师、国产软件适配工程师
🤖 04 无人自主系统板块
培养方向ROS2开发、SLAM导航、飞控系统、强化学习与运动规划
实训内容无人机仿真（Gazebo/PX4）、自主导航算法调试、机械臂控制实验、多机协同任务规划
适配岗位机器人开发工程师、无人系统工程师、ROS2工程师
📊 05 数字化运营与项目管理板块
培养方向AI项目管理、数据驱动运营、招投标与标书撰写、企业数字化咨询
实训内容真实企业AI项目全流程参与、招投标文件撰写实练、数字化方案路演、企业数字化诊断报告
适配岗位AI项目经理、数字化咨询顾问、招投标专员、企业数字化运营专员
Part/05
师资团队与授课方式
01 / 三元师资模式
🎓
校内专业教师
AI、计算机、电子信息领域专任教师及科研团队，理论功底深厚
🏭
企业技术导师
来自麒麟、龙芯、盖特、派宝等合作企业的资深工程师与项目总监
🌟
行业专家
龙芯、麒麟、无人系统等领域知名专家，不定期开展前沿讲座
02 / 多元授课方式
  • 1、
    校内课堂：理论讲解、代码实操、案例分析、算法研讨，学用一体
  • 2、
    企业实训：赴合作企业参与真实项目开发，专项技能训练营、岗位轮岗学习
  • 3、
    产业现场：赴智慧城市示范基地、无人系统实验室实地教学，沉浸式工程实践
  • 4、
    线上教学：配套录播课程与技术文档库，支持课后自主巩固与进阶学习
03 / 学分管理

产业学院所有课程、实训项目均按照学校规定执行学分置换，按学年统一核定学时与学分，与学生原有学业体系无缝衔接，不影响正常毕业、升学及考研。

🚀 多元发展通道
可依托项目作品集参与学科竞赛（大创、挑战杯）、考研升学；优秀学员可参与企业技术研讨与行业标准制定工作
Part/08
管理与考核要求
1
综合考核制度：考核内容包含考勤、专项技术作业、项目实操成果、实训表现、期末综合测评等，考核不合格者取消产业学院学习资格。
2
纪律要求：全体学员需严格遵守考勤制度，不得迟到、旷课、中途退出，保质保量完成各项学习、开发与实践任务。
3
荣誉激励：设置优秀学员奖励机制，学期末评选"人工智能产业学院优秀学员"，颁发证书并优先获得企业内推与实习资格。
